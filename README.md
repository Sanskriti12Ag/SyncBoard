# SyncBoard 🚀

A real-time collaborative whiteboard platform built using React.js, Node.js, Socket.IO, and Tailwind CSS.

## Features

- Real-time collaborative drawing
- Private collaboration rooms
- Join room via room code
- Real-time team chat
- Dark/light theme support
- Save whiteboard as PNG
- Modern SaaS-style UI
- Multi-user synchronization

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Socket.IO Client
- React Router DOM

### Backend
- Node.js
- Express.js
- Socket.IO

## Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
